### Some important notes.
* 在 ```inAnalysis2/src/config/env.js```有幾個需要調整的參數。

| 名稱 | 功能 | 效果 |
| -------- | -------- | -------- |
| signupTime      | 開啟註冊時間     | 需要設定在未來的時間，主畫面才會出現註冊功能     |
|deadline |死線|作業的繳交期限，系統會直接關閉|
